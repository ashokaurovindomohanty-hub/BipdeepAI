Thank you for considering contributing to BipdeepAI! 
## Getting Started
1. Fork the repository on GitHub.
2. Clone your forked repository using `git clone https://github.com/your-username/BipdeepAI.git`.
3. Install dependencies using `pip install -r requirements.txt`.
## Contribution Guidelines
1. Ensure your code is well-documented and follows PEP 8 guidelines.
2. Write unit tests for your code using the `unittest` framework.
3. Use GitHub Issues for discussing bugs and features.
## Pull Request Guidelines
1. Create a new branch for your changes using `git checkout -b your-feature`.
2. Make your changes and commit them using `git commit -m "Your commit message"`.
3. Push your branch to your forked repository using `git push origin your-feature`.
4. Open a Pull Request on the main BipdeepAI repository.
## Code of Conduct
* Be respectful and considerate towards others.
* Avoid conflicts of interest.
* Keep the codebase clean and organized.
