# BipdeepAI
BipdeepAI is a Python project for building a robust AI system.
## Features
* Robust AI modeling
* Efficient data processing
## Requirements
* Python 3.9
* pandas
* sklearn
* unittest
## Installation
1. Clone the repository: `git clone https://github.com/ashokaurovindomohanty-hub/BipdeepAI.git`
2. Install dependencies: `pip install -r requirements.txt`
## Usage
1. Run tests: `python -m unittest tests/test_model.py`
